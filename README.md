# IT Inventory

A tool to monitor all IT devices



## Authors

* [@remusrigo](https://github.com/RemusRigo)
* [@remusrigo](https://sourceforge.net/u/remusrigo/profile)



## Installation

* install a web server (XAMPP)
* go to ./Install and run Create\_DB.bat to create the database and all tables
* default user: admin
* default password: admin
